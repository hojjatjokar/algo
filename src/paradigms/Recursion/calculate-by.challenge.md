# Calculate by

Write a function that gets an array which has 3 item. first and last may integer or calculated to be an integer. middle one will be a operator.

## Examples

### Example 1

- input: `[1, '+', [2, '*', [5, '.', 2]]]`
- output: 7

### Example 2

- input: `[24, '/', [2, '*', [5, '-', 2]]]`
- output: 4
