# flatten

Write a recursive function called flatten which accepts an array of arrays and returns a new array with all values flattened.

## Examples

### Example 1

- input: `[1, 2, 3, [4, 5]]`
- output: `[1, 2, 3, 4, 5]`

### Example 2

- input: `[1, [2, [3, 4], [[5]]]]`
- output: `[1, 2, 3, 4, 5]`
