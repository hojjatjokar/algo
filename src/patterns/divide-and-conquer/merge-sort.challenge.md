# Merge sort

Given an array arr[], its starting position l and its ending position r. Sort the array using merge sort algorithm.

## Examples

### Example 1:

- input: [1, 3, 4, 7, 9]
- output: [1, 3, 4, 7, 9]

### Example 2:

- input: [3,7,4,1,8,0,9]
- output: [0,1,3,4,7,8,9]

### Example 3:

- input: [3,7,4,1,8,0]
- output: [0,1,3,4,7,8]
