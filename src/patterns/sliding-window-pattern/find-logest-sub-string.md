# Find Longest Substring

Write a function called findLongestSubstring, which accepts a string 
and returns the length of the longest substring with all distinct characters.

## Examples
### Example 1
- input: ''
- output: 0

### Example 2
- input: 'rithmschool'
- output: 7

### Example 3
- input: 'thecatinthehat'
- output: 7

### Example 4
- input: 'bbbbbb'
- output: 1
