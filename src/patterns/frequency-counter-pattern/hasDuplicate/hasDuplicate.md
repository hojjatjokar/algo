# Contains Duplicate

Implement a function that accepts an array of items. Determin of the array has any duplicated item.

- input: array of items
- output: true/false

## Case 1

- input: `[1, 2, 3, 1]`
- output: `true`

## Case 2

- input: `[1, 2, 3, 4]`
- output: `false`

## Case 3

- input: `['a', 'b', 'c', 'a']`
- output: `true`

## Case 4

- input: `['a', 'b', 'c', 'd']`
- output: `false`
